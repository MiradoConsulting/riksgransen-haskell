Type signatures

The first line of a function is usually its type signature. All other lines are the implementation:

bigger :: Int -> Int -> Int bigger x y = if x > y then x else y

In a type signature, the last element is the return type, all other elements are inputs. So bigger is a function which accepts two Ints and returns an Int.

Constants are just functions with no inputs, e.g:

port :: Int port = 8080
