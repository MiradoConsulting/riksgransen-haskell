Haskell Teaching Tool

Functions are usually written with a type signature and an implementation, e.g.:

bigger :: Int -> Int -> Int
bigger x y = if x > y then x else y

The last type on the top line is the return type, the others are the inputs.

Exercise: Please write a function abs which returns the absolute value of an integer. You can include the type signature 

abs :: Int -> Int
or leave it out.
