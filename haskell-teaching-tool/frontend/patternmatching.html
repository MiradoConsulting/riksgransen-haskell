If data types are all about grouping data together, then pattern matching is all about breaking it apart.

Here's the definition of Maybe

data Maybe a = Just a
             | Nothing

Here's one way of pattern matching on it (note the case..of keywords):

describe :: Maybe Colour -> String
describe maybeColour =
  case maybeColour of
    Just colour -> "It has a colour."
    Nothing     -> "It does not have a colour!"

Here's another way of pattern matching on it:

describe :: Maybe Colour -> String
describe (Just colour) = "It has a colour."
describe       Nothing = "It does not have a colour!"

Notice how the two possible matches correspond to the data definition at the top.
