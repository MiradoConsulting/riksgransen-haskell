Sometimes you want to pass some state along in a function. It can be convenient to use an extra variable, sometimes called an accumulator, to do so.

Here's an example calculating list length using an accumulator: 

listLength :: Int -> [a] -> Int
listLength acc     [] = acc
listLength acc (x:xs) = listLength (acc+1) xs
It's can be nice to hide the function with the extra parameter as a helper function underneath a top-level function. Here is that: 
listLength :: [a] -> Int
listLength xs = go 0 xs
    where
    go acc     [] = acc
    go acc (x:xs) = go (acc+1) xs

This way you get to use an accumulator but still expose a one-parameter function to your caller.
