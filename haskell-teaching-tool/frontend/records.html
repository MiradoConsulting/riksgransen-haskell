Records

Data types can also be defined using record syntax.

data User = User { getName :: String , getIp :: Ip }

Records are just data types with built-in getters, e.g.:

makeAndPrintUser :: IO () makeAndPrintUser = do -- Create a user let user = User { getName = "John" , getIp = Ip "192.168.0.10" } -- Print the user's name putStrLn (getName user)
