List processing

Write a function listLength to check the size of a list.

A list can either be empty [], or one element followed by a list (x:xs).

Two possible approaches are:

  1. write the function on two lines: listLength [] = ... listLength (x:xs) = ...

  2. write the function on one line: listLength xs = ... using if-then-else and the functions: null :: [a] -> Bool head :: [a] -> a tail :: [a] -> [a]
