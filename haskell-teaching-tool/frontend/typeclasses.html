Type classes

Type classes are a way of adding behaviour to types.

Types which can be shown as a String implement the Show type class, whose declaration looks like:

class Show a where
    show :: a -> String

If you wanted to make Bool showable, your implementation might look like:

instance Show Bool where
    show True  = "True"
    show False = "False"

Derivable Type classes

Many type classes can be derived automatically, for instance Eq (the equality typeclass.)

Here's a type which derives Show and Eq automatically:

data Direction = Up | Down
         deriving (Show, Eq)
